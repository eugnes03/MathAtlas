Definition

Functor

Category Theory · category.tex
A functor $F$ is a map of categories that preserves commutative diagrams. In particular, given two categories, Category $\mathcal{C}$, $\mathcal{D}$, a functor $F : \mathcal{C} \to \mathcal{D}$ satisfies the following properties:
  1. Associates each object $X \in \operatorname{Ob}(\mathcal{C})$ to an object $F(X) \in \operatorname{Ob}(\mathcal{D})$
  2. Associates each morphism $f : X \to Y$ in $\mathcal{C}$ in $\mathcal{C}$ to a morphism $F(f) : F(X) \to F(Y)$ in $\mathcal{D}$ such that the following conditions hold: \begin{enumerate}
  3. $F(\operatorname{id}_X) = \operatorname{id}_{F(X)}$
  4. $F(g \circ f) = F(g) \circ F(f)$ for all morphisms $f : X \to Y$ and $g : Y \to Z$ in $\mathcal{C}$.
\end{enumerate} That is, functors must preserve identity morphisms and composition of morphisms.
